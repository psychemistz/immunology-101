# T Cells & Cell-Mediated Immunity

## Overview

If B cells are the key-value store, T cells are the **process scheduler** of
adaptive immunity. They don't produce antibodies — instead, they orchestrate
responses (helper T cells) or directly kill compromised cells (cytotoxic T
cells). Every adaptive immune response depends on T cell coordination.

This module covers T cell development, subsets, activation, and effector
functions.

---

## T Cell Development in the Thymus

T cells develop in the **thymus** (hence "T"). The thymus enforces brutal
quality control — ~95% of developing thymocytes die.

| Stage | Event | CS Analogy |
|-------|-------|------------|
| Double-negative (DN) | TCR β chain rearrangement | Generating a candidate key |
| Double-positive (DP) | TCR α chain rearrangement; express both CD4 and CD8 | Key pair generation |
| Positive selection | Must bind self-MHC with moderate affinity | Integration test — must work with the API |
| Negative selection | Must NOT bind self-peptide/MHC too strongly | Security audit — reject dangerous inputs |
| Single-positive (SP) | Commits to CD4⁺ or CD8⁺ lineage | Role assignment — scheduler or killer |

*Analogy: the thymus is a CI/CD pipeline — build (rearrangement), test (selection),
deploy (export to periphery). Most builds fail.*

## Major T Cell Subsets

### CD4⁺ Helper T Cells

Helper T cells recognise antigen on **MHC class II** (APCs only) and secrete
cytokines to direct immune responses:

| Subset | Key Cytokines | Target | CS Analogy |
|--------|--------------|--------|------------|
| **Th1** | IFN-γ, TNF-α | Intracellular pathogens (macrophage activation) | Escalation to sysadmin |
| **Th2** | IL-4, IL-5, IL-13 | Helminths, allergy (eosinophil/mast cell activation) | Webhook to specialised service |
| **Th17** | IL-17, IL-22 | Extracellular bacteria, fungi (neutrophil recruitment) | Batch job dispatcher |
| **Tfh** | IL-21 | B cells in germinal centres (antibody help) | Build trigger for CI pipeline |
| **Treg** | IL-10, TGF-β | Immune suppression, tolerance | Rate limiter / circuit breaker |

### CD8⁺ Cytotoxic T Cells (CTLs)

CTLs recognise antigen on **MHC class I** (all nucleated cells) and kill targets
via:

- **Perforin/granzyme pathway** — perforin punches holes; granzymes enter and
  trigger apoptosis. *Analogy: exploit + payload delivery.*
- **Fas/FasL pathway** — death receptor signalling. *Analogy: sending SIGKILL.*

## T Cell Activation — The Two-Signal Model

T cells require **two signals** to activate (prevents accidental firing):

1. **Signal 1**: TCR binds peptide-MHC on an APC. *Analogy: authentication —
   correct credentials.*
2. **Signal 2**: Co-stimulatory molecules (e.g., CD28 on T cell binds B7 on APC).
   *Analogy: authorisation — you have permission.*

Without Signal 2, the T cell becomes **anergic** (unresponsive) — a tolerance
mechanism.

**Signal 3** (cytokine environment) determines the differentiation path
(Th1, Th2, Th17, etc.).

---

## The T Cell Receptor (TCR)

The TCR is a heterodimer (α + β chains, or γ + δ) that recognises a specific
peptide-MHC complex. Unlike antibodies:

- TCRs are **never secreted** — always membrane-bound.
- TCRs recognise peptide **+** MHC together (MHC restriction).
- Diversity is generated by V(D)J recombination (like BCR) but there is **no
  somatic hypermutation**.

*Analogy: the TCR is like an ACL rule — it checks both the payload (peptide) and
the source address (MHC).*

## Bioinformatics Relevance

- **scRNA-seq + TCR-seq**: Paired transcriptome and TCR clonotype analysis
  reveals T cell states and antigen-driven expansion.
- **MHC/HLA typing**: Computational HLA genotyping from WGS/RNA-seq is critical
  for neoantigen prediction and transplant matching.
- **Neoantigen prediction**: Pipelines (pVACtools, NetMHCpan) predict which
  tumour mutations generate MHC-presented peptides targetable by T cells.
- **UMAP/clustering**: T cell subsets (Th1, Th2, Treg, CTL) form distinct
  clusters in scRNA-seq, identifiable by canonical markers and signatures.

## Summary

| Concept | Key Point |
|---------|-----------|
| Thymic development | V(D)J → positive selection → negative selection → export |
| CD4⁺ subsets | Th1, Th2, Th17, Tfh, Treg — each with distinct cytokine profiles |
| CD8⁺ CTLs | Kill via perforin/granzyme or Fas/FasL |
| Activation | Two-signal model: TCR (authentication) + co-stimulation (authorisation) |
| TCR | Membrane-bound, recognises peptide-MHC, no somatic hypermutation |
